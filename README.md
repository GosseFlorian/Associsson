# associsson 
plateforme de gestion de projets et de suivi pédagogique pour les assosiation.

# Prérequis
- Node.js en back-end 
- react js en font-end

## instalation

1. git clone https://github.com/.../associsson
2. cd associsson && npm install
3. cp .env.example .env — renseigner les variables
4. npm run migrate
5. npm run dev → http://localhost:3000

## contribution

- Crée une branche depuis main : git switch -c feature/ma-feature
- Une branche = une feature ou un fix
- PR obligatoire, 1 approbation minimum requise
- Convention de commits : Conventional Commits
